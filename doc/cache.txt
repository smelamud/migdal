Кэш в памяти
============

Кэш в памяти используется для устранения дублирующихся запросов к базе в ходе
генерации одной страницы. Кэш реализован функциями из модуля lib/cache.php.
Элемент из кэша извлекается по 3 параметрам:

Раздел		Таблица		Имя		Значение
------		-------		---		--------

catalog		entries		id записи	Значение поля catalog для
						этой записи.
entry		entries		id записи	Значение поля entry для записи
                                                с указанным id в таблице
						entries.
grp		entries		id записи	Значение поля grp для записи с
						указанным id в таблице entries.
grps            entries         id записи       Массив значений grp из таблицы
                                                entry_grps для entry с
						указанным id
ident		имя таблицы	ident записи	Значение поля id для записи с
						указанным ident в указанной
						таблице.
login           users           login записи    Значение поля id для записи в
                                                таблице users с указанным
						login.
name		entries		id записи	Объект Topic, содержащий все
						для получения имени топика с
						данным id.
obj		entries		id записи	Объект Topic, Posting или
                                                Complain, содержащий полную
						информацию о топике, постинге
						или жалобе с данным id.
track		имя таблицы	id записи	Значение поля track для этой
						записи в указанной таблице.
